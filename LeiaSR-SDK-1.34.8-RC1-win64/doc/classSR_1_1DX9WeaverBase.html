◆ getFrameBuffer()

IDirect3DSurface9 * SR::DX9WeaverBase::getFrameBuffer ( )

Returns the buffer that will be used to create a weaved image. This buffer expects a side-by-side image. When a frame buffer is provided by calling setInputFrameBuffer(), the internally created buffer will be released.

The buffer identifier can be bound using SetRenderTarget to start rendering to it. Rendering to the buffer is identical to normal rendering and often starts with calling Clear.

Returns
IDirect3DSurface9 identifying a buffer generated with CreateTexture
